# django_blog_tutorial
博客项目:

实现功能:

​	1 后台发布文章,管理文章,标签分类等等.

​	2 站点显示博客文章,按时间顺序显示,可以分页显示,分类显示,分标签显示.

​	3 可以对每个文章进行评论,评论按时间顺序显示

​	4 可以进行搜索,搜索结果依据文章标题和文章正文是否包含该关键字.
